# canvas

My scratchbuilt game framework, some basic physics, and a playable asteroids game.

## scripts
run the demo(s)
```
npm start
```

run the tests
```
npm test
```

generate test coverage report
```
npm run coverage
open coverage/lcov-report/index.html
```
